# AIKit Elements Demo App

This is a SwiftUI component gallery app (FluentUI-style) for iterating on **AIKit Elements** UI components and validating Liquid Glass behavior.

## Run

- Open `Examples/AIKitElementsDemoApp/AIKitElementsDemoApp.xcodeproj`
- Select an iOS 26+ simulator (or device) to validate Liquid Glass
- Build & Run

## What this app is for

- A browsable list of components (Conversation, Prompt Input, Tool, Confirmation, etc.)
- Multiple variants per component (idle/streaming/error/approval, etc.)
- A fast place to iterate on visual style before extracting components into an `AIKitElements` SwiftPM target
