# Security Policy

## Supported versions

AIKit is pre-1.0. Security fixes are applied on a best-effort basis to the latest commit on `main`.

## Reporting a vulnerability

Please do not report security vulnerabilities in public GitHub issues.

Send a private report to: **jcfontecha@gmail.com**

Include:

- Affected component(s)
- Reproduction steps
- Impact assessment
- Suggested remediation (if available)

## Response timeline

Maintainer responses are best-effort, with the following targets:

- Initial acknowledgement: within 7 calendar days
- Triage decision: within 14 calendar days
- Mitigation/fix timeline: shared after triage

## Disclosure policy

- Please allow reasonable time for triage and remediation before public disclosure.
- Once fixed, advisories and release notes will document impact and upgrade guidance.
