# Installation

AIKit is a Swift Package targeting:

- iOS 15+
- macOS 12+

## Swift Package Manager (Xcode)

1. In Xcode, open **File → Add Packages…**
2. Add this repository URL.
3. Select the products you need:
   - `AIKit` (umbrella)
   - or `AIKitCore` + a provider (e.g. `AIKitOpenRouter`)

## Swift Package Manager (Package.swift)

Add the dependency and products you want:

```swift
dependencies: [
  .package(url: "…", from: "0.0.0"),
],
targets: [
  .target(
    name: "MyApp",
    dependencies: [
      "AIKit",
      "AIKitOpenRouter",
    ]
  ),
]
```

## API keys

AIKit does not ship “secure key storage”. Provide keys via your provider configuration (e.g. `OpenRouterProviderSettings(apiKey:)`) and store them using your app’s preferred approach (Keychain, MDM, etc.).
