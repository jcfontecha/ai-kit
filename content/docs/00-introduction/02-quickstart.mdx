# Quickstart

This quickstart uses the OpenRouter provider (`AIKitOpenRouter`).

## 1) Create a model

```swift
import AIKit
import AIKitOpenRouter

let openrouter = createOpenRouter(.init(apiKey: "<OPENROUTER_API_KEY>"))
let model = openrouter.chat("openai/gpt-4o-mini")
```

## 2) Generate text (single-step)

```swift
let result = try await generateText(
  model: model,
  prompt: "Write a haiku about Swift concurrency.",
  output: Output.text()
)

print(result.text)
print(result.usage)
```

## 3) Stream text

```swift
let stream = streamText(
  model: model,
  prompt: "Write a short story in three sentences.",
  output: Output.text()
)

for try await delta in stream.textStream {
  print(delta, terminator: "")
}
```

If you want tool calls, approvals, and step boundaries, consume `stream.fullStream` instead of `textStream`. See [streamText](../03-aikit-core/02-stream-text.mdx).
