# Models and providers

AIKitCore is provider-agnostic. Providers implement `LanguageModel` from `AIKitProviders`:

```swift
public protocol LanguageModel: Sendable {
  var id: String { get }
  var capabilities: ModelCapabilities { get }

  func generate(_ request: ModelRequest) async throws -> ModelResponse
  func stream(_ request: ModelRequest) -> AsyncThrowingStream<ModelStreamPart, Error>
}
```

AIKitCore uses this protocol to implement tool loops, output parsing, retries, and streaming event aggregation.

## Built-in providers

- OpenRouter (`AIKitOpenRouter`): `createOpenRouter(...)` / `openrouter(...)`.
- OpenAI (`AIKitOpenAI`): `createOpenAI(...)` / `openai(...)` (surface only; implementations are not yet available).

## Custom providers

If you have an internal gateway or a model with a custom wire format, implement `LanguageModel` and map your provider’s streaming protocol into `ModelStreamPart`. AIKitCore will take it from there.
