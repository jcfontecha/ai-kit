# `generateText`

`generateText` performs a model call and returns a `GenerateTextResult`.

## Basic usage

```swift
let result = try await generateText(
  model: model,
  prompt: "Summarize this in one sentence: …",
  output: Output.text()
)

print(result.text)
```

## Messages instead of prompt

```swift
let result = try await generateText(
  model: model,
  messages: [
    .system("You are concise."),
    .user("Give me three names for a coffee shop."),
  ],
  output: Output.text()
)
```

## Tools (multi-step)

To enable multi-step tool execution, pass tools and increase `stopWhen` beyond the default single step:

```swift
var tools = ToolRegistry()
// tools.register(...)

let result = try await generateText(
  model: model,
  prompt: "Call the tool, then explain the result.",
  tools: tools,
  stopWhen: [Stop.stepCountIs(5)],
  output: Output.text()
)
```

For tool approvals, see [Tool approvals](./05-tool-approvals.mdx).
