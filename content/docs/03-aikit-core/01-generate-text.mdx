# `generateText`

`generateText` performs a model call and returns a `GenerateTextResult`.

## Basic usage

```swift
let ai = AIClient(model: model)
let result = try await ai.generate("Summarize this in one sentence: …")

print(result.text)
```

## Messages instead of prompt

```swift
let ai = AIClient(model: model)
let result = try await ai.generate(messages: [
  .system("You are concise."),
  .user("Give me three names for a coffee shop."),
])
```

## Tools (multi-step)

To enable multi-step tool execution, pass tools and increase `maxSteps` beyond the default single step:

```swift
var tools = ToolRegistry()
// tools.register(...)

let ai = AIClient(model: model, defaults: .init(tools: tools, maxSteps: 5))
let result = try await ai.generate("Call the tool, then explain the result.")
```

For tool approvals, see [Tool approvals](./05-tool-approvals.mdx).
