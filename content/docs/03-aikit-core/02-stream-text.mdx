# `streamText`

`streamText` runs the same core behavior as `generateText`, but exposes streaming views of the response:

- `textStream`: a stream of emitted text deltas (strings)
- `fullStream`: a stream of structured `TextStreamPart` events
- `partialOutputStream`: partial structured output (depends on `OutputSpec.parsePartial`)

## Basic usage (text deltas)

```swift
let ai = AIClient(model: model)
let stream = ai.stream("Write a short poem.")

for try await delta in stream.textStream {
  print(delta, terminator: "")
}
```

## Full event stream (tools, approvals, steps)

```swift
let ai = AIClient(model: model, defaults: .init(tools: tools, maxSteps: 10))
let stream = ai.stream("Use tools if needed.")

for try await part in stream.fullStream {
  switch part {
  case .startStep(let request, _):
    print("step started", request)
  case .toolCall(let call):
    print("tool call", call)
  case .toolResult(let result):
    print("tool result", result)
  case .finish(let finishReason, _, let usage):
    print("done", finishReason, usage)
  default:
    break
  }
}
```

## Await the final output

`StreamTextResult` is also a collector. After the streams finish, you can await the final value:

```swift
let ai = AIClient(model: model)
let stream = ai.stream("Write a short poem.")

let finalText = try await stream.text
let output = try await stream.output
```
