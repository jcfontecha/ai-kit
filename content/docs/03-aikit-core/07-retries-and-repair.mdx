# Retries and repair

AIKit supports:

- `maxRetries`: retry model calls (provider/network failures)
- `repairToolCall`: attempt recovery when a tool call cannot be parsed/decoded

Both are available on `generateText` and `streamText`, and on `Agent` defaults.

Tool call repair is intended to match AI SDK semantics: when parsing fails, you can supply a function that tries to convert the failing tool call into a valid one (e.g. by fixing JSON).

See `ParseToolCallTests.swift` and `Sources/AIKit/Generation/ToolCallRepair.swift`.
