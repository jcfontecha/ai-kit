# AIKit documentation

AIKit is a Swift Package for building AI features in **iOS/macOS client apps**. It aims to match the semantics of Vercel’s AI SDK, tracked against upstream pinned references, while exposing **Swift-forward**, **type-safe** APIs.

Pinned AI SDK parity reference (as of 2026-02-24):  
<https://github.com/vercel/ai/tree/73d5c5920e0fea7633027fdd87374adc9ba49743>
