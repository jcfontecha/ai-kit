# AIKit documentation

AIKit is a Swift Package for building AI features in **iOS/macOS client apps**. It aims to match the semantics of Vercel’s AI SDK (vendored in this repo under `vendored/ai-sdk/`), while exposing **Swift-forward**, **type-safe** APIs.
